ACmix

  • 论文：On the Integration of Self-Attention and Convolution

模型结构

主要过程就是将卷积和自注意力机制拆分成两个部分

  • 卷积

    • 在第一阶段，将输入的特征图沿某个位置的核权重进行线性投影，即( p , q)。这与标准的1 × 1卷积相同。

\[ \text { Stage I: } \tilde{g}_{i j}^{(p, q)}=K_{p, q} f_{i j} \text {, } \]

  • 在第二阶段，投影后的特征图根据核位置进行平移，最后聚合在一起。

\[ \begin{array}{c} \text { Stage II: } g_{i j}^{(p, q)}=\operatorname{Shift}\left(\tilde{g}_{i j}^{(p, q)}, p-\lfloor k / 2\rfloor, q-\lfloor k / 2\rfloor\right) \\ g_{i j}=\sum_{p, q} g_{i j}^{(p, q)} \end{array} \]

  • 自注意力机制 \[ Stage I: q_{i j}^{(l)}=W_{q}^{(l)} f_{i j}, k_{i j}^{(l)}=W_{k}^{(l)} f_{i j}, v_{i j}^{(l)}=W_{v}^{(l)} f_{i j} \] \[ Stage II: g_{i j}=\|_{l=1}^{N}\left(\sum_{a, b \in \mathcal{N}_{k}(i, j)} \mathrm{A}\left(q_{i j}^{(l)}, k_{a b}^{(l)}\right) v_{a b}^{(l)}\right) . \]

经过实验发现卷积操作和自注意力机制操作的第一阶段是相同的，并且发现参数量基本上都花费在了第一阶段，第二阶段不用消耗什么额外的参数，所以可以将两个的第一阶段进行合并，分为两个通道分别进行卷积和自注意力，最后进行合并 \[ F_{\text {out }}=\alpha F_{\mathrm{att}}+\beta F_{\mathrm{conv}} \]
